# HK-APP Development Database

This is the development database for the HK-APP project.

## How to Run

1. Navigate to this directory: `cd hk-db-dev`
2. Start the development container:
   ```bash
   docker-compose up -d
   ```

## Dev Credentials

The database credentials for the development environment:
- **Service Name:** db-dev
- **Container Name:** hk-app-db-dev
- **User:** dev_user
- **Password:** dev_password
- **Database:** hk_app_dev
- **Port:** 5432
